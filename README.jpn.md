## インストール手順
1. [Releases](https://github.com/arakitatsuzou/EhPanda/releases)からipaファイルを取得。
2. [AltStore](https://altstore.io)とかでipaファイルをデバイスにインストール。

このアプリを使うには、デバイスのシステムがiOS・iPadOS 14.0以上であることが必要です。
