## 安装步骤
1. 在[Releases](https://github.com/arakitatsuzou/EhPanda/releases)取得ipa文件。
2. 使用[AltStore](https://altstore.io)这类软件将ipa文件安装到你的设备。

此外，在安装前请确保你的设备系统为iOS/iPadOS 14.0以上。
